# Expense Dataset Management System

## Project Title
Expense Dataset Management System

## Description
This project is a menu-driven Python console application designed to manage and display an expense dataset.
The application allows users to view expense records stored as a list of dictionaries and save the dataset to a file.
It is developed using basic Python concepts covered in class.

## Features
- Stores expense records with 5 attributes
- Displays dataset using menu-driven options
- Uses list and dictionary for data storage
- Saves dataset to a text file
- Simple and user-friendly console interaction

## Concepts Used
- Lists and Dictionaries
- User-defined Functions
- While Loop
- Conditional Statements (if / elif / else)
- String Methods
- File Handling using with open()
- Basic Input/Output

## Instructions to Run the Program
1. Open the file `expense_dataset_project.ipynb` in Jupyter Notebook
2. Run all cells in sequence
3. Use the menu options displayed in the console
4. Enter numeric choices as instructed

## Input Guidelines
- Menu options accept numeric input only
- Enter valid choices such as 1 or 2
- Invalid input will prompt an error message

## File Handling Details
- The dataset is saved to a file named `expense_data.txt`
- Each expense record is written as a dictionary in the file
- File is created using Python file handling methods
